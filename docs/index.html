Creating a Loot Table

Actually creating a loot table is easy! In reality, it's more tedious than anything. Take a look at the template I provided below.

# some-table.yml

alpha: ONE

table:

  • alpha: ONE # alpha can be ONE, TWO, or THREE.
  • table:
    • pool_1:
      • tries: 3
      • loot:
        • stone_sword:
          • weight: 15
          • item:
            • material: STONE_SWORD
        • leather_chestplate:
          • weight: 12
          • item:
            • material: LEATHER_CHESTPLATE
